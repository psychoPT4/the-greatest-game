# Fulfilling Knight
> *"A knight's duty is never fulfilled..."*

## 📖 项目概览 (Project Overview)
《**Fulfilling Knight - Graphic Era**》是一款基于现代 C++ 与高性能硬件加速图形库 **Raylib** 深度构建的硬核 **2D 类银河恶魔城平台动作游戏 (Metroidvania Action Platformer)**。
本项目全面采用 C++20 标准架构开发，摒弃了高度封装的商业化引擎中间件，从零打造了极其精密流畅的底层运动学引擎、定制化多维博弈 AI 决策树，以及跨周期一致性的快照级纯二进制存储管线。

---

## ✨ 核心特色与游戏机制 (Core Features)

### ⚔️ 极致流畅的硬核动作交互
* **动态运动学底盘**：内置自定义欧拉积分重力与流体/地面摩擦阻尼衰减，支持极其顺滑的平地跑酷与空中微调控制。
* **动能冲刺 (Dash)**：消耗耐力进行高速无敌位移，伴随极其惊艳的纯蓝色多重物理拖尾粒子渲染。
* **下劈弹刀 (Pogo Slash)**：在空中垂直下落阶段向下挥剑，若命中敌怪头部或底层地刺陷阱，不仅无伤规避判定，更能借助反弹动能二次跃起并全量重置空中二段跳权限，大幅拓宽跑酷上限。
* **单向跳板下跳**：在空中作战平台按住下键与跳跃，可瞬间离散化穿越单向支撑跳板，接战极度丝滑。

### 🔮 魂系法术与进阶神技体系
* **专注凝聚 (Focus Heal)**：长按施法键，在极佳的纯白能量脉冲与光环包围中消耗 33 点 MP 精准恢复 25 点生命值。
* **灵魂冲击 (Soul Blast)**：短按施法键释放远程灵魂弹幕，对路径目标造成毁灭性贯穿打击。
* **带土式神威虚化 (Kamui Phasing Mode)**：当累计斩杀满 24 只敌怪时自动解锁终极进阶奥义。按 `Q` 键开启空间相位位移，角色进入绝对无敌状态，身躯两侧生成动态插值发散的渐变残影拷贝与公转能量螺旋，可直接穿透敌方高强度弹幕与贴地波。

### 🧠 拒绝套路的史诗级高智商 Boss 战
* 终极领主 **假骑士 (False Knight)** 内部搭载五阶流转状态机，彻底告别机械重复的攻击脚本。
* **12 格动态空间感知**：Boss 实时探知身后的物理长廊深度。面对贴脸猛攻，会极其敏捷地执行后撤大跳拉扯，并在触地瞬间挥动巨锤释放峰值严格锁定为 **1.8 格** 的弧形死神冲击波，强制要求玩家长按大跳越过波峰。
* **死角换位压杀**：若领主连续后撤被逼入墙角，AI 中枢瞬间打破死锁，强行切换为极其刚猛的向前跨顶换位大跳，将硬核动作博弈感推向顶峰。

### 🎨 惊艳的视觉管线呈现
* **动态视差长卷**：深度分层映射平原、深渊与神圣光束，依据镜头移速呈现错落有致的空间景深。
* **加法混合着色矩阵**：完美集成加法混合模式 (`BLEND_ADDITIVE`) 的刀光挥砍切片、动态十字命中暴击爆点及受击顿帧震屏反馈。

## 🎮 操作键位指南 (Controls)

| 键位指令 | 动作反馈 | 消耗/说明 |
| :--- | :--- | :--- |
| **A / D** | 左右平滑移动 | 自动应用地面摩擦与空中风阻 |
| **Shift (长按)** | 极速奔跑模式 | 持续消耗耐力条 |
| **K** | 原地起跳 / 空中二段跳 | 极其跟手的自定义重力轴 |
| **J** | 普通挥砍 / 判定平A | 基础近战伤害 |
| **W + J** | 向上挑击 | 攻击头顶空中的目标 |
| **S + J (半空)**| 向下劈击 (Pogo Slash) | 命中陷阱/敌人反弹并重置二段跳 |
| **S + K (平台)**| 下跳穿越单向跳板 | 快速切入下层战场 |
| **L** | 高速冲刺 (Dash) | 消耗 30 点耐力值 |
| **I (长按 1s)** | 专注回血 (Focus Heal) | 消耗 33 点 MP 恢复 25 点 HP |
| **I (短按)** | 灵魂冲击 (Soul Blast)| 消耗 33 点 MP 释放法术弹幕 |
| **Q** | 开启神威虚化状态 | 需累计击杀 24 只怪物解锁 |
| **F5** | 快速保存游戏快照 | 全量同步至二进制文件 `save.dat` |
| **F9** | 快速载入游戏快照 | 瞬间恢复全量地图与怪物时序轴 |
| **ESC** | 暂停菜单 / 设置控制 | 调整音效与背景音乐全局增益 |

## 💾 存档与数据对齐规范
项目原生内嵌安全沙盒级断点恢复机制，自动校验文件头部标识印印记 (`FKGT`) 及数据块版本协议。支持热重载，完美规避读档时序覆盖与残留实体内存越界。
